

* Example 1

** Source Code 
long day = 24 * 60 * 60 * 1000;
long offset = -8 * 3600 * 1000; // -28800000
long time1 = 0;
long time2 = 17532 * day + offset; 

System.out.println("time1\t" + toFullDate(new Date(time1)));
System.out.println("time2\t" + toFullDate(new Date(time2)));

** Executing Results
$ time    value 
$ time1   1970-01-01 08:00:00
$ time2   2018-01-01 00:00:00


* Example 2

** Source code 
long time = time2 - day*2;
for(int i = 0; i < 52; i ++) { 
    System.out.printf("Week %2d: %s - %s\n", i + 1, 
            toFullDate(new Date(time)), toFullDate(new Date(time + week - 1)));
    time += week;
}

# See at the bottom

// formatted date.
public static String toFullDate(Date date) {
	return new SimpleDateFormat("yyyy-MM-dd HH:mm:ss").format(date);
}

* Example 3
** Source Code
try {
    SimpleDateFormat ft1 = new SimpleDateFormat("yyyy-MM-dd");
    SimpleDateFormat ft2 = new SimpleDateFormat("yyyy-MM-dd hh:mm:ss");
    
    System.out.println(toFullDate(ft1.parse("2018-01-01 10:20:30")));
    System.out.println(toFullDate(ft2.parse("2018-01-01 10:20:30")));
} catch (Exception e) {
}
