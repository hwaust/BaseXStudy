prehand

CMD = command key
ALT = alt key
CTL = control key 
SFT = shift key
TAB = table key
 -  = combined keys, e.g A-B means press key A and key B at the same time.



* Work with windows and tabs
Action          Shortcut
CMD-N           New window
CMD-T           New tab
Shift-CMD-T     Show or hide tab bar
Shift-CMD-\     Show all tabs or exit tab overview
Shift-CMD-N     New command (Execute only once)
Shift-CMD-K     New remote connection
CMD-I           Show or hide Inspector
Shift-CMD-I     Edit title
Option-CMD-I    Edit background color
CMD-Plus (+)    Make fonts bigger
CMD-Minus (â€“)   Make fonts smaller
CMD-`           Next window
CMD-Shift-~     Previous window
CTL-Tab         Next Tab
CTL-Shift-Tab   Previous Tab
CMD-[1-9]       Switch to 1 - 9 tab. 
CMD-D           Split window into two panes
Shift-CMD-D     Close split pane
CMD-W           Close tab
Shift-CMD-W     Close window with more than one tabs
OPT-CMD-W       Close other tabs
OPT-SFT-CMD-W   Close all

CMD-Home        Scroll to top
CMD-End         Scroll to bottom
CMD-PageUp      Page up
CMD-Page Down   Page down
OPT-CMD-PageUp  Line up
OPT-CMD-Page    DownLine down
